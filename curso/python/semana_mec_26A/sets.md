# Sets

## Idea

Eliminar duplicados en estados o valores.

## Micropráctica

```python
estados = ["LED_ON", "LED_OFF", "LED_ON", "BUZZER_ON", "LED_OFF"]
estados_unicos = set(estados)

print(estados_unicos)
```

## Recordar

* Un `set` no conserva duplicados.
