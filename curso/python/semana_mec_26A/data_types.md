# Data Types

## Idea

Reconocer que no todo dato es del mismo tipo.

## Micropráctica

```python
luz = 350              # int
temperatura = 27.5     # float
estado = "activo"      # str
alarma = True          # bool

print(type(luz))
print(type(temperatura))
print(type(estado))
print(type(alarma))
```

### Recuerda

* `int`, `float`, `str`, `bool`.
