# Dictionaries

## Idea

Asociar nombre del sensor con su valor.

## Micropráctica

```python
datos = {
    "luz": 320,
    "temperatura": 27,
    "distancia": 18
}

print(datos)
print("Luz:", datos["luz"])
print("Temperatura:", datos["temperatura"])
```

## Ecuación asociada

$
D = {k_1:x_1,; k_2:x_2,; k_3:x_3}
$

**Objetivo:** representar variables nombradas del sistema.

Donde:

* $k_i$: nombre de la variable.
* $x_i$: valor asociado.
