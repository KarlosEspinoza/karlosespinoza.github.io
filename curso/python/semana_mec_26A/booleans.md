# Booleans

## Idea

Representar estados lógicos del sistema.

## Micropráctica

```python
luz = 250
es_oscuro = luz < 300

print(es_oscuro)
print(type(es_oscuro))
```

## Ecuación asociada

$$
b =
\begin{cases}
\text{True} & \text{si } x < 300 \\
\text{False} & \text{si } x \ge 300
\end{cases}
$$

**Objetivo:** modelar una condición lógica.

Donde:

* $x$: valor de luz.
* $b$: estado booleano.
