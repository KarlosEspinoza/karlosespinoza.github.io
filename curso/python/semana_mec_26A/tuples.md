# Tuples

## Idea

Guardar configuraciones fijas.

## Micropráctica

```python
puertos_validos = ("COM3", "COM4", "/dev/ttyUSB0")
print(puertos_validos)
print(puertos_validos[0])
```

## Recordar

* Una tupla es ordenada pero inmutable.
