# Iterators

## Idea

Entender que una colección puede recorrerse internamente como flujo de datos.

## Micropráctica

```python
lecturas = [100, 200, 300]
it = iter(lecturas)

print(next(it))
print(next(it))
print(next(it))
```

## Recordar

* Qué hay detrás de un `for`.
* Concepto útil para procesamiento de datos.
