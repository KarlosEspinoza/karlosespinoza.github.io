# Operators

### Idea

Usar operadores aritméticos, relacionales y lógicos.

## Micropráctica

```python
luz = 250
temp = 31

print(luz + 50)
print(temp > 30)
print(luz < 300 and temp > 25)
```

## Recordar

* `+`, `-`, `*`, `/`
* `<`, `>`, `==`
* `and`, `or`, `not`
