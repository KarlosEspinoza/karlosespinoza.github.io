# Strings

## Idea

Manipular mensajes que vienen del sistema.

## Micropráctica

```python
mensaje = "LUZ:350"
print(mensaje)

partes = mensaje.split(":")
print(partes)
print("Variable:", partes[0])
print("Valor:", partes[1])
```
