# Casting

## Idea

Convertir tipos de datos.

## Micropráctica

Simular que Arduino envía un valor como texto.

```python
dato_serial = "350"
luz = int(dato_serial)

print(luz)
print(type(luz))
```

## Ecuación asociada

$
x_{numérico} = \text{cast}(x_{texto})
$

**Objetivo:** transformar datos crudos en datos utilizables.
